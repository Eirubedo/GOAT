# GOAT

A repository to store R code and raw data for research.

## Repository Structure

```
GOAT/
├── R/                  # R scripts for data analysis
│   └── R2 GA.R      # Main analysis script
├── data/
│   └── raw/            # Raw, unprocessed data files
└── README.md
```

## Usage

1. Place raw data files in `data/raw/`.
2. Run the R scripts in `R/` to process and analyse the data.

## Requirements

- R (>= 4.0.0)
